# Lotto Number Generator

## Overview

A simple web application to generate random lottery numbers. The application will feature a clean and modern interface where users can click a button to get a new set of lottery numbers.

## Project Outline

### Style and Design
*   **Layout:** Centered, single-column layout.
*   **Colors:** A modern color palette with a dark background and vibrant colors for the numbers.
*   **Typography:** Clear and readable fonts.
*   **Components:**
    *   A main container for the application.
    *   A title.
    *   A display area for the generated numbers, with each number appearing as a colored ball.
    *   A "Generate" button.
*   **Effects:**
    *   Subtle noise texture for the background.
    *   Soft, deep shadows for a "lifted" look on the number balls and button.
    *   "Glow" effect on the button.

### Features
*   Generate 6 unique random numbers between 1 and 45.
*   Display the numbers in a visually appealing way.
*   A button to trigger the number generation.
*   Use of a Web Component (`lotto-ball`) to display each number.

## Current Task

### Plan
1.  **`index.html`**: Set up the basic HTML structure with a title, a container for the lottery numbers, and a button.
2.  **`style.css`**: Add styles to create a modern and visually appealing interface.
3.  **`main.js`**: Implement the JavaScript logic to generate unique random numbers and display them on the screen using a custom web component.
